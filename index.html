My tinyFileSystem

My tinyFileSystem is based off of the UFS Hybrid Filesystem. The first FileExtent block address is stored in the Inode, and other FileExtent block addresses are stored in the InodeIndex (stores all other FileExtent addresses). Features include CBC encryption to secure the file system, READ and READ_WRITE protection, timestamp for creation, change, access, and modify time, and terminal output debug mode.
